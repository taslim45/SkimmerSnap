Skimmer is an audio-based text skimming application in Android. Research study shows that Skimmer can improve motion sickness for people who wants to consume text material while on-the-go to 1 in 6 people. 

A demo of Skimmer is available here https://youtu.be/b62MvfLORmw 

Skimmer tech-stack:
- Java
- JavaScript (touch interactions and audio-playback)
- HTML & CSS
